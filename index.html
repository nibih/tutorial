Git & Github

One of the most essential tools is git. here is the link provided in an anchor tag: github

Git is what we call "version control" it allows you to host your projects but still keep track of what changes are made. you can even make branches if you want to have different versions running. Github is just a site where you can store all the data remotely.

a good video about git^

CodePen

CodePen is a good tool to view your code and the output side by side in real time. it is really useful when you are prototyping.

Visual Studio Code

This is the leading "text editor" for coding right now. it is very versatile as it can be used to write a lot of different programming languages. it is made by microsoft and is available through their website

    VS Code has some extensions that can help make your life easier, such as:

  • prettier: makes your code all neat and readable

  • live server: kinda like CodePen but offline

  • Bracket Pair Colorizer: helps you keep track of which brackets belong together

  • auto rename tag: basically autocomplete for your HTML

  • CSS peek: a tool to see what the CSS is for a certain element

  • code spell checker: explains itself
